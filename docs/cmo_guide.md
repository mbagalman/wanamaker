# What Wanamaker Tells Your CMO

This page will explain Wanamaker outputs for marketing executives and budget decision-makers.

## Planned Sections

### Executive Summary

What changed, what the model believes, and what should be treated carefully.

### Channel Contributions

How much each channel appears to have contributed, with uncertainty shown plainly.

### ROI

Return estimates, including when the model cannot estimate a channel reliably.

### Trust Card

The short version of whether the model is fit for decision-making.

### Scenario Comparison

How to compare user-supplied budget plans without treating the model as an optimizer.

### Experiment Advisor

Which channels need experimental validation because the model cannot learn enough from
observational data alone.

### What Wanamaker Cannot Tell You

Wanamaker should say when data is insufficient, when channels are weakly identified, and
when a recommendation would require evidence the model does not have.
