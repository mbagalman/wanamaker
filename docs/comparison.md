# Comparison to Other Tools

This page will compare Wanamaker honestly against other MMM tools. The goal is clarity,
not salesmanship.

## Tools To Cover

- Robyn
- Meridian
- Recast
- PyMC-Marketing

## Comparison Pattern

Each section will cover:

- What the other tool does well
- Who it is best for
- Where Wanamaker fits better
- Where the other tool is the better choice

## Current Positioning

Wanamaker is intended for mid-market teams that need local-first Bayesian MMM, clear
diagnostics, refresh accountability, and decision-oriented reports without building a
full data science program.
